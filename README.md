# Let's Talk About Autism

An autism awareness and resource platform, built to give families, caregivers, and communities clear information, support, and ways to get involved.

## Sections

- **About** — mission and background
- **Services** — support offerings
- **Resources** — reference material for families and caregivers
- **Blog** — articles and updates
- **Get Involved** — volunteering and community participation
- **Contact**

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router) + TypeScript
- Tailwind CSS, [Framer Motion](https://www.framer.com/motion/), [lucide-react](https://lucide.dev/) icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.
